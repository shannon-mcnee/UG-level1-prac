Welcome to Level One Psychology Practical Support!


The aim of this resource is to provide you with hints, tips, and support based on questions from previous Level One students.

Setting Up introduces you to the R environment, different file types, and teaches you how to resolve common set up errors.

Introduction to Tidyverse and Data Visualisation introduces you to the R package tidyverse, highlighting how we can use the ‘Wickham Six’ to make data analysis enjoyable and produce beautiful data visualisations.


Installing R on your own device

If you are using your own device and have yet to install R and R studio, please use this guide to help you get started.
